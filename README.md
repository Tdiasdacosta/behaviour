NatureScope (Version Complète)

Ce projet est une mini‑application Vue.js permettant d'explorer des merveilles naturelles.

Fonctionnalités

Navigation entre plusieurs faits naturels
Flip du texte/titre au clic
Interface modernisée (glassmorphism)
Design sombre inspiré de la nature
 Structure du projet

index.html — point d'entrée
css/style.css — styles visuels
data/data.js — données (titres / descriptions / images)
js/script.js — logique Vue.js
